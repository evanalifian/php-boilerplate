# PHP Boilerplate

A starter template to build website using native PHP and MVC pattern implementation.

## How To Clone

```bash
git clone https://github.com/evanalifian/php-boilerplate.git <optional-project-name>
composer install
```

## Tech Stacks

- **PHP:** Programming language.
- **Bootstrap:** CSS Framework.
